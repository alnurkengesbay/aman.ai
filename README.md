# Aman AI: Интеллектуальная платформа для диагностики и реабилитации нейродегенеративных заболеваний

## Аннотация

Данный программный комплекс представляет собой интеграционную платформу, объединяющую шесть специализированных модулей на основе искусственного интеллекта для комплексной диагностики и реабилитации пациентов с нейродегенеративными заболеваниями. Платформа реализует мультимодальный подход к анализу медицинских данных, включающий обработку нейровизуализационных исследований, биосигналов, генетической информации и лабораторных показателей.

**Ключевые слова:** нейродегенеративные заболевания, машинное обучение, глубокое обучение, медицинская диагностика, нейровизуализация, телемедицина.

**URL:** https://amanai.kz

## 1. Введение

### 1.1. Актуальность исследования

Нейродегенеративные заболевания представляют одну из наиболее значимых проблем современной медицины. По данным Всемирной организации здравоохранения, распространённость деменции и связанных с ней состояний неуклонно возрастает, что обусловливает необходимость разработки эффективных методов ранней диагностики и мониторинга состояния пациентов.

### 1.2. Цель и задачи

**Цель работы:** разработка интегрированной программной платформы для автоматизированной диагностики нейродегенеративных заболеваний с использованием методов машинного обучения.

**Задачи:**
1. Разработка модуля анализа КТ/МРТ снимков головного мозга на основе свёрточных нейронных сетей
2. Реализация системы мониторинга физиологических показателей посредством IoT-устройств
3. Создание модуля психометрического тестирования с автоматизированным анализом результатов
4. Интеграция методов анализа генетических данных (AlphaFold, ESMFold)
5. Разработка системы анализа лабораторных биомаркеров крови
6. Реализация модуля нейрореабилитации с использованием компьютерного зрения

## 4. Ролевая модель системы

Система реализует разграничение доступа на основе ролевой модели (RBAC):

### 4.1. Пациент (PATIENT)
- Загрузка результатов медицинских исследований
- Прохождение психометрических тестов
- Просмотр результатов автоматизированного анализа
- Получение врачебных заключений

### 4.2. Врач (DOCTOR)
- Просмотр данных прикреплённых пациентов
- Верификация результатов AI-диагностики
- Формирование медицинских заключений и рекомендаций
- Назначение дополнительных исследований

### 4.3. Администратор (ADMIN)
- Управление учётными записями пользователей
- Конфигурирование внешних сервисов
- Мониторинг работоспособности системы
- Управление правами доступа

### 5.1. Требования к среде выполнения

### 5.3. Развёртывание

#### 5.3.1. Установка зависимостей

```bash
# Клиентская часть
npm install

# Серверная часть
cd backend
pip install -r requirements.txt
```

#### 5.3.2. Конфигурация окружения

```bash
# Переменные окружения (.env)
DATABASE_URL="postgresql://user:password@localhost:5432/amanai"
AUTH_SECRET="криптографически стойкий ключ"
AUTH_URL="https://amanai.kz"
```

#### 5.3.3. Инициализация базы данных

```bash
# Применение схемы
npx prisma db push

# Миграции (при разработке)
npx prisma migrate dev
```

#### 5.3.4. Запуск сервисов

```bash
# Frontend (порт 3000)
npm run dev

# Backend (порт 8000)
cd backend && uvicorn app.main:app --reload --port 8000
```

## 7. Интеграция внешних сервисов

### 7.1. Протокол взаимодействия

Внешние диагностические модули интегрируются посредством механизма iframe с передачей контекста пользователя через URL-параметры.

```typescript
// Формат URL сервиса
https://service.example.com/?userId={userId}&token={jwt}

// Формат ответа сервиса
POST https://amanai.kz/api/services/results
Content-Type: application/json

{
  "userId": "string",
  "serviceType": "CT_MRI | IOT | QUESTIONNAIRE | GENETICS | BLOOD | REHABILITATION",
  "result": { /* структурированные данные */ },
  "confidence": 0.0-1.0,
  "findings": ["string"]
}
```

### 7.2. Требования к пользовательскому интерфейсу сервисов

Для обеспечения визуальной согласованности внешние модули должны соответствовать единой дизайн-системе:

- Цветовая схема: монохромная (градации серого)
- Типографика: Manrope (основной), JetBrains Mono (моноширинный)
- Компоненты: скруглённые углы (border-radius: 1rem)
